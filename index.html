Welcome to GitHub Pages.

This is an Android application development project. Now-a-days it is a growing need to Pay User's shopping bill, restaurant bill, any kind of shopping bill through user's mobile devices. So for that I have generated this repository which reads User's credit/debit card information easily and give them satisfaction of paying bill through user's android devices. After this, you can check out the code branch:

$ cd saurabh9890/Credit-Debit-Card-Reader/src/
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
